# Microsserviço de Cadastro de Usuários — Backend da Aplicação: Come-come

## Sobre o Projeto
Este repositório faz parte do projeto da disciplina Projeto de Desenvolvimento de Software e implementa o microsserviço de cadastro de usuários do backend da aplicação Come-come — um aplicativo mobile que auxilia usuários a analisarem informações nutricionais de produtos alimentícios e compararem produtos em diversos aspectos (como calorias, açúcares, gorduras, entre outros).

O backend é desenvolvido em Spring Boot, seguindo a arquitetura de microsserviços, e será consumido pela aplicação mobile.

## Objetivo
O objetivo deste microsserviço é gerenciar o cadastro, autenticação e manutenção de dados de usuários da plataforma, garantindo segurança, escalabilidade e integração fluida com os demais serviços da aplicação.

## Tecnologias Utilizadas

- Java 21
- Spring Boot 3.5.6
- Spring Data JPA
- PostgreSQL
- Maven
